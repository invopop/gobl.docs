Period represents two dates with a start and finish.

## Schema ID

`https://gobl.org/draft-0/cal/period`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| Label | `label` | string | Label is a short description of the period. |
| Start | `start` | [cal.cal.Date](/draft-0/cal/cal._date) | Start indicates when this period starts. |
| End | `end` | [cal.cal.Date](/draft-0/cal/cal._date) | End indicates when the period ends, and must be after the start date. |
