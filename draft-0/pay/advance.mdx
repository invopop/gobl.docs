Advance represents a single payment that has been made already, such as a deposit on an intent to purchase, or as credit from a previous invoice which was later corrected or cancelled.

## Schema ID

`https://gobl.org/draft-0/pay/advance`

## Attributes

| Property   | Type                                           | Required | Calculated | Description                                                                          |
|------------|------------------------------------------------|----------|------------|--------------------------------------------------------------------------------------|
| `uuid`     | [GOBL::UUID::UUID](/draft-0/uuid/uuid)         | false    | false      | Unique identifier for this advance.                                                  |
| `date`     | [GOBL::Cal::Date](/draft-0/cal/date)           | false    | false      | When the advance was made.                                                           |
| `ref`      | String                                         | false    | false      | ID or reference for the advance.                                                     |
| `grant`    | Boolean                                        | false    | false      | If this "advance" payment has come from a public grant or subsidy, set this to true. |

| `amount`   | GOBL::Num::Amount                              | true     | false      | How much was paid.                                                                   |
| `currency` | [GOBL::Currency::Code](/draft-0/currency/code) | false    | false      | If different from the parent document's base currency.                               |
