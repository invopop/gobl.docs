Digest defines a structure to hold a digest value including the algorithm used to generate it.

## Schema ID

`https://gobl.org/draft-0/dsig/digest`

## Attributes

| Property | Type   | Required | Calculated | Description                                                                                     |
|----------|--------|----------|------------|-------------------------------------------------------------------------------------------------|
| `alg`    | String | true     | false      | Algorithm stores the algorithm key that was used to generate the value.                         |
| `val`    | String | true     | false      | Value contains the Hexadecimal representation of the resulting hash generated by the algorithm. |
