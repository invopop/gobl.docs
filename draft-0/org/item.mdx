Item is used to describe a single product or service.

## Schema ID

`https://gobl.org/draft-0/org/item`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| UUID | `uuid` | string | Universally Unique Identifier. |
| Ref | `ref` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | Primary reference code that identifies this item. Additional codes can be provided in the 'identities' property. |
| Key | `key` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Special key used to classify the item sometimes required by some regimes. |
| Name | `name` | string | Brief name of the item |
| Identities | `identities` | array of [org.org.Identity](/draft-0/org/org._identity) | List of additional codes, IDs, or SKUs which can be used to identify the item. They should be agreed upon between supplier and customer. |
| Description | `description` | string | Detailed description of the item. |
| Images | `images` | array of [org.org.Image](/draft-0/org/org._image) | Images associated with the item. |
| Currency | `currency` | [currency.currency.Code](/draft-0/currency/currency._code) | Currency used for the item's price. |
| Price | `price` | [num.num.Amount](/draft-0/num/num._amount) | Base price of a single unit to be sold. Must be either zero or positive. |
| Alternative Prices | `alt_prices` | array of [currency.currency.Amount](/draft-0/currency/currency._amount) | AltPrices defines a list of prices with their currencies that may be used as an alternative to the item's base price. |
| Unit | `unit` | [org.org.Unit](/draft-0/org/org._unit) | Unit of measure. |
| Country of Origin | `origin` | [l10n.l10n.ISOCountryCode](/draft-0/l10n/l10n._i_s_o_country_code) | Country code of where this item was from originally. |
| Extensions | `ext` | [tax.tax.Extensions](/draft-0/tax/tax._extensions) | Extension code map for any additional regime specific codes that may be required. |
| Meta | `meta` | [cbc.cbc.Meta](/draft-0/cbc/cbc._meta) | Additional meta information that may be useful |

## Key Values

| Value | Description |
|-------|-------------|
| `goods` | Goods |
| `services` | Services |
