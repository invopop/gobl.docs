DocumentRef is used to describe an existing document or a specific part of it's contents.

## Schema ID

`https://gobl.org/draft-0/org/document-ref`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| UUID | `uuid` | string | Universally Unique Identifier. |
| Schema | `schema` | string | Schema of the referenced document if different from that of the parent. |
| Type | `type` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Type of the document referenced according to the defined schema or that of the parent document. |
| Issue Date | `issue_date` | [cal.cal.Date](/draft-0/cal/cal._date) | IssueDate reflects the date the document was issued. |
| Series | `series` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | Series the referenced document belongs to. |
| Code | `code` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | Source document's code or other identifier. |
| Currency | `currency` | [currency.currency.Code](/draft-0/currency/currency._code) | Currency used in the document, if different from the parent's currency. |
| Lines | `lines` | array of integer | Line index numbers inside the document, if relevant. |
| Identities | `identities` | array of [org.org.Identity](/draft-0/org/org._identity) | List of additional codes, IDs, or SKUs which can be used to identify the document or its contents, agreed upon by the supplier and customer. |
| Period | `period` | [cal.cal.Period](/draft-0/cal/cal._period) | Tax period in which the referred document had an effect required by some tax regimes and formats. |
| Reason | `reason` | string | Human readable description on why this reference is here or needs to be used. |
| Description | `description` | string | Additional details about the document. |
| Stamps | `stamps` | array of [head.head.Stamp](/draft-0/head/head._stamp) | Seals of approval from other organizations that may need to be listed. |
| URL | `url` | string | Link to the source document. |
| Tax | `tax` | [tax.tax.Total](/draft-0/tax/tax._total) | Tax total breakdown from the original document in the provided currency. Should only be included if required by a specific tax regime or addon. |
| Payable | `payable` | [num.num.Amount](/draft-0/num/num._amount) | Payable is the total amount that is payable in the referenced document. Only needed for specific tax regimes or addons. This may also be used in some scenarios to determine the proportion of the referenced document that has been paid, and calculate the remaining amount due and taxes. |
| Extensions | `ext` | [tax.tax.Extensions](/draft-0/tax/tax._extensions) | Extensions for additional codes that may be required. |
| Meta | `meta` | [cbc.cbc.Meta](/draft-0/cbc/cbc._meta) | Meta contains additional information about the document. |
