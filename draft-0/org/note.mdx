Note represents a free text of additional information that may be added to a document.

## Schema ID

`https://gobl.org/draft-0/org/note`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| UUID | `uuid` | string | Universally Unique Identifier. |
| Key | `key` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Key specifying subject of the text |
| Code | `code` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | Code used for additional data that may be required to identify the note. |
| Source | `src` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Source of this note, especially useful when auto-generated. |
| Text | `text` | string | The contents of the note |
| Meta | `meta` | [cbc.cbc.Meta](/draft-0/cbc/cbc._meta) | Additional information about the note |
| Extensions | `ext` | [tax.tax.Extensions](/draft-0/tax/tax._extensions) | Extension data |
