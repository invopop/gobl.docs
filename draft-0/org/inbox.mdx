Inbox is used to store data about a connection with a service that is responsible for potentially receiving copies of GOBL envelopes or other document formats defined locally.

## Schema ID

`https://gobl.org/draft-0/org/inbox`

## Attributes

| Property | Type                                   | Required | Calculated | Description                                                        |
|----------|----------------------------------------|----------|------------|--------------------------------------------------------------------|
| `uuid`   | [GOBL::UUID::UUID](/draft-0/uuid/uuid) | false    | false      | Unique ID. Useful if inbox is stored in a database.                |
| `key`    | [GOBL::CBC::Key](/draft-0/cbc/key)     | true     | false      | Type of inbox being defined.                                       |
| `role`   | [GOBL::CBC::Key](/draft-0/cbc/key)     | false    | false      | Role assigned to this inbox that may be relevant for the consumer. |
| `name`   | String                                 | false    | false      | Human name for the inbox.                                          |
| `code`   | String                                 | true     | false      | Actual Code or ID that identifies the Inbox.                       |
