Image describes a logo or photo that represents an entity.

## Schema ID

`https://gobl.org/draft-0/org/image`

## Properties

| Property      | Type                                | Req. | Description                                                                                                                           |
|---------------|-------------------------------------|------|---------------------------------------------------------------------------------------------------------------------------------------|
| `uuid`        | [uuid/UUID](/draft-0/uuid/uuid)     |      | Unique ID of the image                                                                                                                |
| `label`       | string                              |      | Label to help identify the image.                                                                                                     |
| `url`         | string                              |      | URL of the image                                                                                                                      |
| `data`        | string                              |      | As an alternative to the URL and only when the source data is small, like an SVG, the raw data may be provided using Base64 encoding. |
| `mime`        | string                              |      | Format of the image.                                                                                                                  |
| `description` | string                              |      | Details of what the image represents.                                                                                                 |
| `alt`         | string                              |      | Alternative text if the image cannot be shown.                                                                                        |
| `height`      | integer                             |      | Height of the image in pixels.                                                                                                        |
| `width`       | integer                             |      | Width of the image in pixels.                                                                                                         |
| `digest`      | [dsig/Digest](/draft-0/dsig/digest) |      | Digest can be used to ensure the image contained at the URL is the same one as originally intended.                                   |
