Person represents a human, and how to contact them electronically.

## Schema ID

`https://gobl.org/draft-0/org/person`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| UUID | `uuid` | string | Universally Unique Identifier. |
| Label | `label` | string | Label can be used to identify the person in a given context in a single language, for example "Attn", "Contact", "Responsible", etc. |
| Key | `key` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Key used to identify the role of the person inside the context of the object. |
| Name | `name` | [org.org.Name](/draft-0/org/org._name) | Complete details on the name of the person. |
| Role | `role` | string | Role or job title of the responsibilities of the person within an organization. |
| Identities | `identities` | array of [org.org.Identity](/draft-0/org/org._identity) | Set of codes used to identify the person, such as ID numbers, social security, driving licenses, etc. that can be attributed to the individual. |
| Postal Addresses | `addresses` | array of [org.org.Address](/draft-0/org/org._address) | Regular post addresses for where information should be sent if needed. |
| Email Addresses | `emails` | array of [org.org.Email](/draft-0/org/org._email) | Electronic mail addresses that belong to the person. |
| Telephone Numbers | `telephones` | array of [org.org.Telephone](/draft-0/org/org._telephone) | Regular phone or mobile numbers |
| Avatars | `avatars` | array of [org.org.Image](/draft-0/org/org._image) | Avatars provider links to images or photos or the person. |
| Meta | `meta` | [cbc.cbc.Meta](/draft-0/cbc/cbc._meta) | Data about the data. |
