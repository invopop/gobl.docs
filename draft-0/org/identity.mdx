Identity is used to define a code for a specific context.

## Schema ID

`https://gobl.org/draft-0/org/identity`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| UUID | `uuid` | string | Universally Unique Identifier. |
| Label | `label` | string | Optional label useful for non-standard identities to give a bit more context. |
| Scope | `scope` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Scope defines the context in which this identity is meant to be used. |
| Country | `country` | [l10n.l10n.ISOCountryCode](/draft-0/l10n/l10n._i_s_o_country_code) | Country from which the identity was issued. |
| Key | `key` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Uniquely classify this identity using a key instead of a Type. |
| Type | `type` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | The type of Code being represented and usually specific for a particular context, country, or tax regime, and cannot be used alongside the key. |
| Code | `code` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | The actual value of the identity code. |
| Description | `description` | string | Description adds details about what the code could mean or imply |
| Extensions | `ext` | [tax.tax.Extensions](/draft-0/tax/tax._extensions) | Ext provides a way to add additional information to the identity. |

## Scope Values

| Value | Description |
|-------|-------------|
| `tax` | Tax |
| `legal` | Legal |
