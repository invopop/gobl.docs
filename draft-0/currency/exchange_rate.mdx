ExchangeRate contains data on the rate to be used when converting amounts from the document's base currency to whatever is defined.

## Schema ID

`https://gobl.org/draft-0/currency/exchange-rate`

## Attributes

| Property   | Type                                           | Required | Calculated | Description                                                        |
|------------|------------------------------------------------|----------|------------|--------------------------------------------------------------------|
| `currency` | [GOBL::Currency::Code](/draft-0/currency/code) | true     | false      | ISO currency code this rate represents.                            |
| `amount`   | GOBL::Num::Amount                              | true     | false      | How much is 1.00 of this currency worth in the documents currency. |
