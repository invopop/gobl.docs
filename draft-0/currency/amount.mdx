An Amount represents a monetary value in a specific currency.

## Schema ID

`https://gobl.org/draft-0/currency/amount`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| Label | `label` | string | Label allows for additional information to be added to the currency Amount that may be useful. |
| Currency | `currency` | [currency.currency.Code](/draft-0/currency/currency._code) | Code defines the currency for this amount. |
| Value | `value` | [num.num.Amount](/draft-0/num/num._amount) | Value is the amount in the currency. |
