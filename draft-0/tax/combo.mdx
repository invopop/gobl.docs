Combo represents the tax combination of a category code and rate key.

## Schema ID

`https://gobl.org/draft-0/tax/set#/$defs/Combo`

## Attributes

| Property    | Type                                 | Required | Calculated | Description                                                                                               |

| `cat`       | [GOBL::CBC::Code](/draft-0/cbc/code) | true     | false      | Tax category code from those available inside a region.                                                   |
| `rate`      | [GOBL::CBC::Key](/draft-0/cbc/key)   | false    | false      | Rate within a category to apply.                                                                          |
| `percent`   | GOBL::Num::Percentage                | false    | true       | Percent defines the percentage set manually or determined from the rate key (calculated if rate present). |
