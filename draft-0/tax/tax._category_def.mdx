CategoryDef contains the definition of a general type of tax inside a region.

## Schema ID

`https://gobl.org/draft-0/tax/regime-def#/$defs/tax.CategoryDef`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| Code | `code` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | Code to be used in documents |
| Name | `name` | [i18n.i18n.String](/draft-0/i18n/i18n._string) | Short name of the category to be used instead of code in output |
| Title | `title` | [i18n.i18n.String](/draft-0/i18n/i18n._string) | Human name for the code to use for titles |
| Description | `desc` | [i18n.i18n.String](/draft-0/i18n/i18n._string) | Useful description of the category. |
| Retained | `retained` | boolean | Retained when true implies that the tax amount will be retained by the buyer on behalf of the supplier, and thus subtracted from the invoice taxable base total. Typically used for taxes related to income. |
| Informative | `informative` | boolean | Informative when true implies that the tax amount will be calculated and reported but will not affect the invoice totals. Typically used for taxes that are embedded in the base amount or don't impact the final payable amount. |
| Keys | `keys` | array of [tax.tax.KeyDef](/draft-0/tax/tax._key_def) | Specific tax definitions inside this category. |
| Rates | `rates` | array of [tax.tax.RateDef](/draft-0/tax/tax._rate_def) | Rates defines the set of rates that can be used with this category. |
| Extensions | `extensions` | array of [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Extensions defines a list of extension keys that may be used or required as an alternative or alongside choosing a rate for the tax category. Every key must be defined in the Regime's extensions table. |
| Map | `map` | [cbc.cbc.CodeMap](/draft-0/cbc/cbc._code_map) | Map defines a set of regime specific code mappings. |
| Sources | `sources` | array of [cbc.cbc.Source](/draft-0/cbc/cbc._source) | List of sources for the information contained in this category and most importantly, an official source where the tax rates can be reviewed and updated in the future. |
| Extensions | `ext` | [tax.tax.Extensions](/draft-0/tax/tax._extensions) | Extension key-value pairs that will be copied to the tax combo if this category is used. |
| Meta | `meta` | [cbc.cbc.Meta](/draft-0/cbc/cbc._meta) | Meta contains additional information about the category that is relevant for local frequently used formats. |
