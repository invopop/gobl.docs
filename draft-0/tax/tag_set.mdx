TagSet defines a set of tags and their descriptions that can be used for a specific schema in the context of a Regime or Addon.

## Schema ID

`https://gobl.org/draft-0/tax/tag-set`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| Schema | `schema` | string | Schema that the tags are associated with. |
| List | `list` | array of [cbc.Definition](/draft-0/cbc/definition) | List of tags for the schema |
