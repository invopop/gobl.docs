RateValueDef contains a percentage rate or fixed amount for a given date range.

## Schema ID

`https://gobl.org/draft-0/tax/regime-def#/$defs/tax.RateValueDef`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| Extensions | `ext` | [tax.tax.Extensions](/draft-0/tax/tax._extensions) | Only apply this rate if one of the tags is present in the invoice. Tags []cbc.Key `json:"tags,omitempty" jsonschema:"title=Tags"` Ext map of keys that can be used to filter to determine if the rate applies. |
| Since | `since` | [cal.cal.Date](/draft-0/cal/cal._date) | Date from which this value should be applied. |
| Percent | `percent` | [num.num.Percentage](/draft-0/num/num._percentage) | Percent rate that should be applied |
| Surcharge | `surcharge` | [num.num.Percentage](/draft-0/num/num._percentage) | An additional surcharge to apply. |
| Disabled | `disabled` | boolean | When true, this value should no longer be used. |
