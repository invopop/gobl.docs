RateTotal contains a sum of all the tax rates in the document with a matching category and rate.

## Schema ID

`https://gobl.org/draft-0/tax/total#/$defs/tax.RateTotal`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| Country | `country` | [l10n.l10n.TaxCountryCode](/draft-0/l10n/l10n._tax_country_code) | Country code override when issuing with taxes applied from different countries, it'd be very strange to mix rates from different countries, but in theory this would be possible. |
| Key | `key` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Tax key if supported by the category. |
| Extensions | `ext` | [tax.tax.Extensions](/draft-0/tax/tax._extensions) | If the rate is defined with extensions, they'll be used to group by also. |
| Base | `base` | [num.num.Amount](/draft-0/num/num._amount) | Base amount that the percentage is applied to. |
| Percent | `percent` | [num.num.Percentage](/draft-0/num/num._percentage) | Percentage of the rate. Will be nil when taxes are **exempt**. |
| Surcharge | `surcharge` | [tax.tax.RateTotalSurcharge](/draft-0/tax/tax._rate_total_surcharge) | Surcharge applied to the rate. |
| Amount | `amount` | [num.num.Amount](/draft-0/num/num._amount) | Total amount of rate, excluding surcharges |
