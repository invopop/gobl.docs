Identity stores the details required to identify an entity for tax purposes.

## Schema ID

`https://gobl.org/draft-0/tax/identity`

## Properties

| Title        | Property  | Type                                           | Req. | Description                                                                                                                                                  |
|--------------|-----------|------------------------------------------------|------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| UUID         | `uuid`    | string                                         |      | Unique universal identity code for this tax identity.                                                                                                        |
| Country Code | `country` | [l10n.CountryCode](/draft-0/l10n/country_code) | true | ISO country code for Where the tax identity was issued.                                                                                                      |
| Type         | `type`    | [cbc.Key](/draft-0/cbc/key)                    |      | Type is set according to the requirements of each regime, some have a single tax document type code, others require a choice to be made.                     |
| Code         | `code`    | [cbc.Code](/draft-0/cbc/code)                  |      | Normalized code shown on the original identity document.                                                                                                     |
| Meta         | `meta`    | [cbc.Meta](/draft-0/cbc/meta)                  |      | Additional details that may be required.                                                                                                                     |
| Zone         | `zone`    | [l10n.Code](/draft-0/l10n/code)                |      | DEPRECATED. Zone was removed 2024-03-14 in favour of using tax tags and extensions with local data when required. Maintained here to support data migration. |
