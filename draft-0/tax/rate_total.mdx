RateTotal contains a sum of all the tax rates in the document with a matching category and rate.

## Schema ID

`https://gobl.org/draft-0/tax/total#/$defs/RateTotal`

## Properties

| Title     | Property    | Type                                                        | Req. | Description                                                               |
|-----------|-------------|-------------------------------------------------------------|------|---------------------------------------------------------------------------|
| Key       | `key`       | [cbc.Key](/draft-0/cbc/key)                                 |      | Optional rate key is required when grouping.                              |
| Ext       | `ext`       | [tax.Extensions](/draft-0/tax/extensions)                   |      | If the rate is defined with extensions, they'll be used to group by also. |
| Base      | `base`      | [num.Amount](/draft-0/num/amount)                           | true | Base amount that the percentage is applied to.                            |
| Percent   | `percent`   | [num.Percentage](/draft-0/num/percentage)                   |      | Percentage of the rate. Will be nil when taxes are **exempt**.            |
| Surcharge | `surcharge` | [tax.RateTotalSurcharge](/draft-0/tax/rate_total_surcharge) |      | Surcharge applied to the rate.                                            |
| Amount    | `amount`    | [num.Amount](/draft-0/num/amount)                           | true | Total amount of rate, excluding surcharges                                |
