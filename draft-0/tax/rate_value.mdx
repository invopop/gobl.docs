RateValue contains a percentage rate or fixed amount for a given date range.

## Schema ID

`https://gobl.org/draft-0/tax/regime#/$defs/RateValue`

## Attributes

| Property    | Type                                     | Required | Calculated | Description                                      |

| `zones`     | [GOBL::L10n::Code](/draft-0/l10n/code)[] | false    | false      | Only use this value if one of the zones matches. |
| `since`     | [GOBL::Cal::Date](/draft-0/cal/date)     | false    | false      | Date from which this value should be applied.    |
| `percent`   | GOBL::Num::Percentage                    | true     | false      | Percent rate that should be applied              |
| `surcharge` | GOBL::Num::Percentage                    | false    | false      | An additional surcharge to apply.                |
| `disabled`  | Boolean                                  | false    | false      | When true, this value should no longer be used.  |
