A CatalogueDef contains a set of re-useable extensions, scenarios, and validators that can be used by addons or tax regimes.

## Schema ID

`https://gobl.org/draft-0/tax/catalogue-def`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
|  | `key` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Key defines a unique identifier for the catalogue. |
|  | `name` | [i18n.i18n.String](/draft-0/i18n/i18n._string) | Name is the name of the catalogue. |
|  | `description` | [i18n.i18n.String](/draft-0/i18n/i18n._string) | Description is a human readable description of the catalogue. |
|  | `extensions` | array of [cbc.cbc.Definition](/draft-0/cbc/cbc._definition) | Extensions defines all the extensions offered by the catalogue. |
