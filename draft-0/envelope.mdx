Envelope wraps around a document adding headers and digital signatures.

## Schema ID

`https://gobl.org/draft-0/envelope`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| JSON Schema ID | `$schema` | string | Schema identifies the schema that should be used to understand this document |
| Header | `head` | [head.head.Header](/draft-0/head/head._header) | Details on what the contents are |
| Document | `doc` | [schema.schema.Object](/draft-0/schema/schema._object) | The data inside the envelope |
| Signatures | `sigs` | array of [dsig.dsig.Signature](/draft-0/dsig/dsig._signature) | JSON Web Signatures of the header |
