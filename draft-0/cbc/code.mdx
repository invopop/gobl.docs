Alphanumerical text identifier with upper-case letters and limits on using special characters or whitespace to separate blocks.

## Schema ID

`https://gobl.org/draft-0/cbc/code`


## Validation Rules

| Code | Field | Test | Description |
| ---- | ----- | ---- | ----------- |
| <code>GOBL-CBC-CODE-01</code> |  | length between 0 and 64 | codes must be no longer than 64 characters |
| <code>GOBL-CBC-CODE-02</code> |  | matches ^[A-ZÑa-z0-9]+([\.\-\:/,_\& ]?[A-ZÑa-z0-9]+)*$ | codes must only contain letters, numbers, and optionally separated by .-:/,_& or space |
