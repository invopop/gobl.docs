KeyDefinition defines properties of a key that is specific for a regime.

## Schema ID

`https://gobl.org/draft-0/cbc/key-definition`

## Properties

| Title       | Property  | Type                                                        | Req. | Description                                                                                                |
|-------------|-----------|-------------------------------------------------------------|------|------------------------------------------------------------------------------------------------------------|
| Key         | `key`     | [cbc.Key](/draft-0/cbc/key)                                 | true | Actual key value.                                                                                          |
| Name        | `name`    | [i18n.String](/draft-0/i18n/string)                         | true | Short name for the key.                                                                                    |
| Description | `desc`    | [i18n.String](/draft-0/i18n/string)                         |      | Description offering more details about when the key should be used.                                       |
| Meta        | `meta`    | [cbc.Meta](/draft-0/cbc/meta)                               |      | Meta defines any additional details that may be useful or associated with the key.                         |
| Codes       | `codes`   | array of [cbc.CodeDefinition](/draft-0/cbc/code_definition) |      | Codes describes the list of codes that can be used alongside the Key, for example with identities.         |
| Keys        | `keys`    | array of [cbc.KeyDefinition](/draft-0/cbc/key_definition)   |      | Keys is used instead of codes to define a further sub-set of keys that can be used alongside this one.     |
| Pattern     | `pattern` | string                                                      |      | Pattern is used to validate the key value instead of using a fixed value from the code or key definitions. |
| Code Map    | `map`     | [cbc.CodeMap](/draft-0/cbc/code_map)                        |      | Map helps map local keys to specific codes, useful for converting the described key into a local code.     |
