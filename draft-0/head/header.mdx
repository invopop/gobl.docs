Header defines the metadata of the body.

## Schema ID

`https://gobl.org/draft-0/head/header`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| UUID | `uuid` | string | Unique UUIDv1 identifier for the envelope. |
| Digest | `dig` | [dsig.dsig.Digest](/draft-0/dsig/dsig._digest) | Digest of the canonical JSON body. |
| Stamps | `stamps` | array of [head.head.Stamp](/draft-0/head/head._stamp) | Seals of approval from other organisations that can only be added to non-draft envelopes. |
| Links | `links` | array of [head.head.Link](/draft-0/head/head._link) | Links provide URLs to other resources that are related to this envelope and unlike stamps can be added even in the draft state. |
| Tags | `tags` | array of string | Set of labels that describe but have no influence on the data. |
| Meta | `meta` | [cbc.cbc.Meta](/draft-0/cbc/cbc._meta) | Additional semi-structured information about this envelope. |
| Notes | `notes` | string | Any information that may be relevant to other humans about this envelope |
