FuelAccountItem provides the details of a fuel purchase.

## Schema ID

`https://gobl.org/draft-0/regimes/mx/fuel-account-balance#/$defs/cfdi.FuelAccountItem`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| Type | `type` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | Type of fuel (one of `c_ClaveTipoCombustible` codes, maps to `TipoCombustible`). |
| Unit | `unit` | [org.org.Unit](/draft-0/org/org._unit) | Reference unit of measure used in the price and the quantity (maps to `Unidad`). |
| Name | `name` | string | Name of the fuel (maps to `NombreCombustible`). |
| Price | `price` | [num.num.Amount](/draft-0/num/num._amount) | Base price of a single unit of the fuel without taxes (maps to `ValorUnitario`). |
