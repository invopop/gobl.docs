LineCharge represents an amount added to the line, and will be applied before taxes.

## Schema ID

`https://gobl.org/draft-0/bill/line#/$defs/bill.LineCharge`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| Key | `key` | [cbc.cbc.Key](/draft-0/cbc/cbc._key) | Key for grouping or identifying charges for tax purposes. A suggested list of keys is provided, but these are for reference only and may be extended by the issuer. |
| Code | `code` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | Reference or ID for this charge defined by the issuer |
| Reason | `reason` | string | Text description as to why the charge was applied |
| Base | `base` | [num.num.Amount](/draft-0/num/num._amount) | Base for percent calculations instead of the line's sum |
| Percent | `percent` | [num.num.Percentage](/draft-0/num/num._percentage) | Percentage of base or parent line's sum |
| Quantity | `quantity` | [num.num.Amount](/draft-0/num/num._amount) | Quantity of units to apply the charge to when using the rate instead of the line's quantity. |
| Unit | `unit` | [org.org.Unit](/draft-0/org/org._unit) | Unit to associate with the quantity when using the rate. |
| Rate | `rate` | [num.num.Amount](/draft-0/num/num._amount) | Rate defines a price per unit to use instead of the percentage. |
| Amount | `amount` | [num.num.Amount](/draft-0/num/num._amount) | Fixed or resulting charge amount to apply (calculated if percent present). |
| Extensions | `ext` | [tax.tax.Extensions](/draft-0/tax/tax._extensions) | Extension codes that apply to the charge |
