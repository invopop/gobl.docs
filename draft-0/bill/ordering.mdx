Ordering provides additional information about the ordering process including references to other documents and alternative parties involved in the order-to-delivery process.

## Schema ID

`https://gobl.org/draft-0/bill/ordering`

## Properties

| Title | Property | Type | Description |
|-------|----------|------|-------------|
| Code | `code` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | Identifier assigned by the customer or buyer for internal routing purposes. |
| Identities | `identities` | array of [org.org.Identity](/draft-0/org/org._identity) | Any additional Codes, IDs, SKUs, or other regional or custom identifiers that may be used to identify the order. |
| Cost | `cost` | [cbc.cbc.Code](/draft-0/cbc/cbc._code) | Buyer accounting reference cost code associated with the document. |
| Period | `period` | [cal.cal.Period](/draft-0/cal/cal._period) | Period of time that the invoice document refers to often used in addition to the details provided in the individual line items. |
| Buyer | `buyer` | [org.org.Party](/draft-0/org/org._party) | Party who is responsible for issuing payment, if not the same as the customer. |
| Seller | `seller` | [org.org.Party](/draft-0/org/org._party) | Seller is the party liable to pay taxes on the transaction if not the same as the supplier. |
| Issuer | `issuer` | [org.org.Party](/draft-0/org/org._party) | Issuer represents a third party responsible for issuing the invoice, but is not responsible for tax. Some tax regimes and formats require this field. |
| Projects | `projects` | array of [org.org.DocumentRef](/draft-0/org/org._document_ref) | Projects this invoice refers to. |
| Contracts | `contracts` | array of [org.org.DocumentRef](/draft-0/org/org._document_ref) | The identification of contracts. |
| Purchase Orders | `purchases` | array of [org.org.DocumentRef](/draft-0/org/org._document_ref) | Purchase orders issued by the customer or buyer. |
| Sales Orders | `sales` | array of [org.org.DocumentRef](/draft-0/org/org._document_ref) | Sales orders issued by the supplier or seller. |
| Receiving Advice | `receiving` | array of [org.org.DocumentRef](/draft-0/org/org._document_ref) | Receiving Advice. |
| Despatch Advice | `despatch` | array of [org.org.DocumentRef](/draft-0/org/org._document_ref) | Despatch advice. |
| Tender Advice | `tender` | array of [org.org.DocumentRef](/draft-0/org/org._document_ref) | Tender advice, the identification of the call for tender or lot the invoice relates to. |
