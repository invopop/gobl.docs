The Colombian tax regime is based on the DIAN (Dirección de Impuestos y Aduanas Nacionales)
specifications for electronic invoicing.

## Base Details

| Key | Value |
| --- | ----- |
| Country Code | <code>CO</code> |
| Currency | <code>COP</code> |
| Base Time Zone | <code>America/Bogota</code> |

## Tax Categories

| Code | Name | Title |
| ---- | ---- | ----- |
| <code>VAT</code> | VAT | Value Added Tax |
| <code>IC</code> | IC | Consumption Tax |
| <code>ICA</code> | ICA | Industry and Commerce Tax |
| <code>RVAT</code> | ReteIVA | Retención en la fuente por el Impuesto al Valor Agregado |
| <code>RICA</code> | ReteICA | Retención en la fuente por el Impuesto de Industria y Comercio |
| <code>RR</code> | Retefuente | Retención en la fuente por el Impuesto de la Renta |

### VAT Rates

| Key | Name | Percents | Description |
| --- | ---- | -------- | ----------- |
| <code>zero</code> | Zero Rate | 0.0% |  |
| <code>standard</code> | Standard Rate | 19.0% |  |
| <code>reduced</code> | Reduced Rate | 5.0% |  |

## Correction Definitions

This tax regime supports auto-generation of corrective invoices
or credit and debit notes.

### Invoice Types

The types of invoices that can be created with a preceding definition:
- <code>credit-note</code>

### Stamp Keys

Stamp keys from the previous invoice that need to be referenced:
- <code>dian-cude</code>

### Correction Methods

| Key | Name | Extras |
| --- | ---- | ------ |
| <code>partial</code> | Partial refund | <code>dian:1</code> |
| <code>revoked</code> | Revoked | <code>dian:2</code> |
| <code>discount</code> | Discount | <code>dian:3</code> |
| <code>price-adjustment</code> | Adjustment | <code>dian:4</code> |
| <code>other</code> |  | <code>dian:5</code> |
